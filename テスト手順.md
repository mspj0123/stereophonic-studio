# PoC テスト手順 — ブラウザ内で分離 → 立体音響で再生

この `d-browser-app/player.html` は、**曲をアップロードせず、あなたのブラウザの中だけで**
Demucs AI が楽器ごと（vocals / drums / bass / other）に分離し、そのまま立体音響で
再生できる——という「本命構成」の中核（PoC）です。

> ⚠ これはまず「本当にブラウザ内分離が動くか」を確かめるための検証版です。
> ライブラリ画面・共有ライブラリ・アップロードボタン等の全体UIは次の段階で作ります。

---

## 動かし方（ローカルで確認する場合）

`file://` で直接開くと動きません（Service Worker が使えないため）。**localhost で開いてください。**

1. ターミナルで、このフォルダに移動してサーバーを起動：
   ```
   cd /home/manabu/projects/stereophonic_sound/d-browser-app
   python3 -m http.server 8000
   ```
   （このプロジェクトの Claude セッションなら、プロンプトに
   `! cd /home/manabu/projects/stereophonic_sound/d-browser-app && python3 -m http.server 8000`
   と打てばこの場で起動できます）

2. **Chrome または Edge**（WebGPU対応）で次を開く：
   ```
   http://localhost:8000/player.html
   ```
   - 開いた瞬間、ページが**一度自動リロード**します（正常です。`coi-serviceworker` が
     クロスオリジン分離を有効化するため）。

3. 上部「🎤 音源を分離」パネルで：
   - バッジが **WebGPU（GPU・高速）** か **WebAssembly（CPU・低速）** を表示します
   - **「＋ 分離する曲を選ぶ」** で音源ファイル（mp3 / wav / m4a 等）を選ぶ
   - 初回だけ**分離モデル（約172MB）のダウンロード**が走ります（プログレス表示あり）
   - その後**分離が始まります**（曲の長さと端末性能により数十秒〜数分）

4. 分離が終わると、下の「音源リスト」に **vocals / drums / bass / other** が自動で追加されます：
   - **「▶ 再生」** を押す
   - **「立体音響」を ON** にすると、各パートが3D空間に配置されて聞こえます
   - 位置パッドで各パートの位置をドラッグして変えられます

## 確認してほしいこと（フィードバックください）

1. バッジは WebGPU / WebAssembly のどちらでしたか？（速度に直結します）
2. 分離は最後まで完了しましたか？ **かかった時間**（画面に秒数が出ます）
3. 分離の**音質**は許容範囲ですか？（vocals がちゃんと声だけになっているか等）
4. 立体音響で**3Dに聞こえましたか**？
5. エラーが出たら、その**赤いメッセージ全文**と、使ったブラウザ名を教えてください。
   （ブラウザの「開発者ツール → Console」に出るエラーもあれば助かります）

## 既知の注意点（正直な制約）

- **端末性能に依存**：4分の曲で、WebGPU対応PCなら1〜数分、CPU(WASM)のみだと10〜30分ほど。
  まずは**短い曲（1〜2分）**で試すのがおすすめです。
- **メモリ**：モデル172MB＋処理で数GB使います。低メモリ端末・スマホは失敗することがあります。
- **対応ブラウザ**：Chrome / Edge 推奨（WebGPU）。Safari / Firefox は WASM で動く場合もありますが未確認。
- **初回のみ172MBのダウンロード**（以後はブラウザにキャッシュされます）。

## 使っている部品（すべて無料・カード不要）

- 分離エンジン：`vendor/demucs-web/`（timcsy/demucs-web、**MITライセンス**。Demucs v4 を ONNX Runtime Web で実行）
- 分離モデル：Hugging Face Hub 上の公開ONNX（約172MB／カード不要でホスト・取得可）
- クロスオリジン分離：`coi-serviceworker.js`（gzuidhof、**MIT**。GitHub Pages 等の静的ホストで
  SharedArrayBuffer を有効化）
- 3Dプレイヤー：元の立体音響プレイヤー（`original/stereophonic_sound_2.html`）をそのまま使用
